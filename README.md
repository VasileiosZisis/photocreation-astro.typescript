# Studio Photocreation Astro Rebuild

This repo is the learning-focused Astro rebuild of the existing Vite React
portfolio site.

The goal is to rebuild the current site in small, understandable steps before
doing any larger redesign work.

## Recommended Stack

- Astro
- TypeScript
- Plain CSS using BEM naming
- Astro Content Collections for manually edited gallery data
- Cloudinary for image and video delivery
- PhotoSwipe for the gallery lightbox
- `@astrojs/sitemap`
- Netlify static deployment
